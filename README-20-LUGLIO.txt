Avvio locale
1. Apri un terminale in questa cartella.
2. Esegui: python -m http.server 8000
3. Apri nel browser: http://localhost:8000

È possibile anche aprire index.html direttamente, ma un piccolo server locale
riproduce meglio il comportamento dell'hosting finale e dei file multimediali.

Verifica automatica (se Node.js è installato)
node tests/verify-unlocks.cjs

Pubblicazione
Carica l'intero contenuto di questa cartella sul servizio che ospita il sito,
mantenendo invariata la struttura delle cartelle. Il file iniziale è index.html.
